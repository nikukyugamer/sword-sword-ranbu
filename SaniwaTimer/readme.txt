==========================
さにわタイマー for Windows
==========================
どこかで見たようなタイマーです（当然、非公式です）。
素材の提供を快く承諾して下さった@rokuremix様、原Webアプリ開発者の@millefeui様には多大なる感謝と御礼を申し上げます。

＜使い方＞
とにかく右クリックをしましょう。何とかなるはずです。
何とかならなかったらご連絡下さい。
任意文字列で使える漢字は「教育漢字1006字」のみです（「付録」参照）。
あとあまりに長い任意文字にするときっと変になります。

＜使用フォント＞
「白舟極太楷書教漢」を使用させて頂いております。
ありがとうございます。
http://www.hakusyu.com/download/kyokan.html

＜今後の予定＞
ドキュメントの充実。
設定の保存や本編との連動などいろいろやりたいことはあります。
あとアイコンを何とかしたい…。

不具合報告や要望などをお待ちしております。

＜改版履歴＞
Version 1.0.0.0 (2015/05/03)
時間が00:00:00になった時に任意の音声ファイルを再生できるようにした（デフォルトはシステム情報音）。
「設定時間」を保持したまま再設定に入れるようにした。

表示文字列に任意の文字列を設定できるようにした。
右クリックのメニュー項目を全般的に追加・修正した。

Version 0.1.0.1 (2015/02/26)
表示文字を「〆切」「待機列」「定時」から選べるようにしました。
フォントの変更をしました。
その他細かい内部処理の修正をしました。

Version 0.0.0.6 (2015/02/25)
背景画像が表示されないという最凶バグを修正しました。ほんとごめんなさい。
